# Code of Conduct

## Expectations

Participation in the Codictate project should remain respectful, constructive, and professional.

Examples of expected behavior:

- Be direct, but stay respectful.
- Assume good intent and focus on the technical issue.
- Accept feedback and course-correct quickly.
- Keep discussions relevant to improving the project.

Examples of unacceptable behavior:

- Harassment, personal attacks, or discriminatory language
- Trolling, intimidation, or deliberate disruption
- Publishing private information without permission

## Scope

This applies to project spaces such as issues, pull requests, discussions, and other public channels connected to the repository.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct.

For serious concerns, contact the maintainer through the security contact listed in `SECURITY.md`.
